# ZCity DarkRP NPC Shop

Standalone NPC shop addon for Garry's Mod.

Features:
- Buy weapons and entities by using an NPC
- Uses DarkRP money at runtime
- Improved UI with player shop and manager view
- ULX-only trader management for `admin` and `superadmin`
- Admin catalog editor with persistent items
- Auto-detected lists for weapons, armor, ammo, and attachments from loaded server content
- Persistent NPC name setting
- Admin commands to spawn and remove persistent shop NPCs

Commands:
- `zcity_darkrp_shop`
- `zcity_darkrp_shop_admin`
- `zcity_darkrp_shop_spawn_npc`
- `zcity_darkrp_shop_remove_npc`

Data files:
- `data/zcity_darkrp_shop/items.json`
- `data/zcity_darkrp_shop/npcs/<map>.json`
